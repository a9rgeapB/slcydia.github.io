window 10 专业版 slmgr激活

发表于 2019-11-10 更新于 2021-11-10 分类于 其他

1、右键开始按钮，在快捷菜单点击命令提示符(管理员)，或者windows PowerShell(管理员);

2、复制这个命令：slmgr.vbs /upk, 在命令窗口鼠标右键，会自动粘贴此命令，按回车确定，卸载之前的密钥，弹出窗口显示“成功地卸载了产品密钥”;【如果提示找不到密钥，不用管，直接跳到下一步】

3、打开命令窗口，复制这个命令slmgr /ipk W269N-WFGWX-YVC9B-4J6C9-T83GX，在命令窗口鼠标右键，会自动粘贴，按回车键执行，安装win10专业版密钥；

4、复制这个命令：slmgr /skms kms.chinancce.com，鼠标右键粘贴此命令，按回车执行，弹出窗口提示：“密钥管理服务计算机名称成功地设置为kms.chinancce.com”，kms.chinancce.com是kms服务器，网络上非常多这种服务器，如果这个不行，可以自行搜索kms服务器，可用的kms激活服务器有哪些;

5、复制这个命令：slmgr /ato，右键自动粘贴此命令，按回车执行，此时将弹出窗口提示：“成功地激活了产品”;

6、要查看激活状态，继续执行命令slmgr /xpr，回车，kms激活都是180天，到期继续执行上面操作续期。

群晖docker安装hexo部署GitHub

发表于 2019-11-08 更新于 2021-11-11 分类于 Hexo

docker安装hexo

打开docker注册表，搜索Hexo并下载

待下载完成后在「映像」标签中双击刚刚下载的映像「spurin/hexo」，点击「高级设置」按钮

映射文件夹（docker/hexo）装载路径（/app）

设置端口，本地端口号根据个人偏好设定

在环境中可以填入自己的 Github 用户名及邮箱，以便生成有效的 SSH Key。

点击应用按钮，之后点击下一步按钮，确认信息填写正确，建立容器，可以勾选完成向导后运行容器

等待容器安装成功并正常运行之后，可以在浏览器中输入 http://群晖 IP:4000，可以看到 Hexo 生成的网站

GitHub 创建 Github Pages

首先创建GitHub pages,登录GitHub，点击右上角「＋」，选择新建仓库。

然后给自己的Repositories取一个名字，注意：名称格式最好为：用户名.github.io

连接GitHub Pages
下一步是连接 Github Pages，打开放置 Hexo 文件的文件夹，找到/.ssh 路径下的 id_rsa.pub 文件，用文本编辑器打开，复制 SSH Key

打开 Github 个人设置，找到「SSH and GPG keys」，新建一个密钥，将密钥粘贴。

在群晖docker终端机中输入 ssh -T git@github.com，若可以看到自己的 Github 用户名，证明连接成功。
